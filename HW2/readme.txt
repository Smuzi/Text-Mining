in order to run the algorithm normally type:
python run.py

you can add number of word clusters and document clusters as follows:
python run.py -wc=150 -dc=75

average time 30-40 seconds


in order to run with different cluster sizes:
python runClusterSize.py

time: 10-12 minutes


in order to run with different thresholds:
python runCriteria.py

time: 2-3 minutes



important files:

usmadja-test-clusters - is the test document clusters.
can run it again with runTest.py (will rewrite the file)

report.pdf - containing full report
